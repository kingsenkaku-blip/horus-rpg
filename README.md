# horus-rpg

1980〜1990年代風のコマンド式RPGプロトタイプです。TypeScript + Vite で実装しています。

## 起動方法

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

## 操作方法

- タイトル画面: クリックで開始
- フィールド画面: 地面をクリックすると目的地へ自動移動
- クリックした地面は青色ハイライト表示
- NPCに接触: 会話メッセージ表示
- 敵に接触: コマンドバトル開始
- バトルコマンド:
  - たたかう: 実装済み
  - まほう: 未実装（メッセージ表示）
  - どうぐ: 未実装（メッセージ表示）
  - にげる: 実装済み

## 現在できること

- タイトル / フィールド / 会話 / バトル / ゲームオーバーの状態遷移
- タイルベースマップ（壁、床、草、道）
- 通行可能・不可能判定
- プレイヤー移動、NPC・敵シンボルとの接触判定
- ターン制戦闘（攻撃・被ダメージ・勝利・敗北）
- HP / 攻撃 / 防御 / 経験値 / レベルの基本ステータス
- メッセージウィンドウ
- スプライト未指定時の「デバッグ」プレースホルダー表示

## GitHub Pages デプロイ

- `.github/workflows/deploy-pages.yml` で `main` ブランチ push 時に自動デプロイします（lockfile 不要のため workflow は `npm install` を使用）。
- GitHub の `Settings > Pages` で **Build and deployment: GitHub Actions** を選んでください。
- Vite の `base` は Actions 実行時のみ `/horus-rpg/` に切り替わる設定です。
