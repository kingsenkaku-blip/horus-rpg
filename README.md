# horus-rpg

1980〜1990年代風の、フィールド移動 + コマンド選択式バトルを持つブラウザRPGプロトタイプです。TypeScript + Vite で実装し、外部画像アセットには依存していません。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで表示されたローカルURLを開くと起動します。

本番ビルド確認:

```bash
npm run build
```

## 操作方法

- 矢印キー / WASD: フィールド移動、バトルコマンド選択
- マップをクリック: クリックしたタイルをハイライトし、そこへ自動移動
- Enter / Space: 決定、メッセージ送り、NPCとの会話
- Escape / Backspace: キャンセル、会話終了、バトルで1階層戻る

## 現在できること

- タイトル画面からゲーム開始
- タイルベースのテストマップを移動
- 平原の上端から別マップ「月蝕の古道」へ移動
- 平原の右端から新マップ「頌帰の庭」へ移動
- 「頌帰の庭」で頌歌の碑、帰源の碑の順に調べると奥への道が開通
- 奥マップ「蒼月の奥殿」で「伝説の剣士レイディア」と戦闘
- 「蒼月の奥殿」の右端から半壊した「カムジンの村」へ移動
- カムジンの村では「蹂躙埜戦姫」に襲われた痕跡と決戦前準備を配置
- クリックしたタイルのハイライトと自動移動
- 壁・水などの通行不可タイル
- NPCへの会話
- 右側の宝箱を開けると「魔法剣・アイリス / ソリッドブラッド」が解放
- 1マップ目の右上で決定すると隠し武器「熾天使の剣＋」が解放
- ランダムに決まるエンカウント歩数に達するとターン制バトル開始
- 固定敵「その辺のヤンキー」への接触でターン制バトル開始
- 別マップ中央の「最強吸血鬼(仮)」との戦闘前会話
- 「最強吸血鬼(仮)」は戦闘開始時に「闇統べる貴公子」を得て、1ターンに1〜3回行動
- 「伝説の剣士レイディア」は戦闘前・戦闘後シナリオを持ち、5つの武具に応じて行動が変化
- コマンド「戦闘 / 魔法 / 道具 / 逃げる」
- 「戦闘」から「テンペストアタック / デバッグ斬」を選択
- 「魔法」から「テンペストウェイブ / デバッグビーム」を選択
- 技が増えた場合、攻撃・魔法画面で上下入力すると表示範囲がスクロール
- 「その辺のヤンキー」を倒すと「フラクチャーブレイド / ディスラプターレイ」が解放
- カムジンの村の武器架から「ミョニュスブレイド / ハイパースラッシュ」が解放
- カムジンの村の魔導具箱から「ドミナスグレイル / スノウ・マン / 風神の乱舞」が解放
- カムジンの村の石碑で、蹂躙埜戦姫の即死斬撃ギミック予告を確認
- 攻撃・魔法の簡易バトルエフェクト表示
- HP、攻撃力、防御力、経験値、レベルの表示と更新
- 勝利時の経験値獲得と簡単なレベルアップ
- HP 0 によるゲームオーバー
- レトロRPG風の黒背景、白文字、枠線付きUI
- スプライト未指定エンティティのデバッグプレースホルダー表示
